# Photo Gallery Web App

A responsive photo gallery web application built using React and Vite.  
Users can search images and mark images as favourites.

## GitHub Repository
https://github.com/Piyush-Singh-01/Photo-Gallary-web-app

## Project link
https://photo-gallary-web-app.vercel.app/

## Tech Stack
- React.js
- Tailwind CSS
- Vite

## Features
- Image gallery display
- Search photos
- Add/remove favourite images
- Responsive design
